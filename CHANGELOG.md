# What's New

All notable changes to Sitku are recorded here. Newest first.
Edit this file each release — the app shows the top entry to users automatically.

## [2026.7.4] — 2026-07-04

### Added
- **🧠 Dynamic Second Brain (Dataview Queries):** Type `:::query` in any note to aggregate incomplete tasks, note directories, and workspace analytics in real-time. Click any task to jump directly to its source note!
- **🔄 Live Draft Sync:** Unsaved editor drafts are now instantly evaluated by embedded queries without waiting to save.

### Fixed
- **⚡ Zero-Flicker File Management:** Deleting or moving folders and notes is now instant and ultra-smooth, with zero loading skeletons or screen flashing.
- **🍏 Instant macOS Reopening:** Closing the window with the red traffic light button on Mac and clicking the Dock icon now reopens your workspace instantly (0ms) where you left off—no cold reload required!

## [2026.7.3] — 2026-07-03

### Fixed
- App no longer gets stuck on the loading screen / white-screens on launch.
- Tab, sidebar, and toolbar icons are visible again.

### Added
- **Auto-update** — you're notified the moment a new version is ready, with a one-click Restart to install it.

### Changed
- Cleaner tab menu: the three-dot button is gone — right-click a tab for the same actions (Close, Split, Rename, …).
